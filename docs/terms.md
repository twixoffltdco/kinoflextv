Используя сайт KinoBox.tv (далее — «Сервис»), вы подтверждаете, что ознакомились с настоящими условиями и соглашаетесь их соблюдать. Если вы не согласны с условиями — пожалуйста, покиньте сайт.

## 2. Характер сервиса

KinoBox.tv является агрегатором — технической оболочкой, которая встраивает видеоплееры сторонних платформ через публичные API и iframe. Сервис не хранит, не загружает и не распространяет видеофайлы напрямую. Весь видеоконтент размещён на серверах третьих сторон.
