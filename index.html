Введи ID Кинопоиска — получи готовый код

СПОСОБ 1

Через kinobox.js

Самый гибкий способ. Один скрипт — много плееров на странице.

<div id="player"></div>

<script src="https://kinobox.tv/kinobox.js"></script>
<script>
  kinobox('#player', {
    search: { kinopoisk: 301 }
  });
</script>
СПОСОБ 2

Через <iframe>

Проще некуда — работает даже в конструкторах сайтов.

<iframe
  src="https://kinobox.tv/embed/#301"
  width="100%"
  height="500"
  frameborder="0"
  allowfullscreen
  allow="autoplay; fullscreen"
></iframe>

📋 Параметры kinobox()

ПараметрТипОписание
selectorstringCSS-селектор контейнера плеера
search.kinopoisknumberID фильма/сериала на Кинопоиске
search.imdbstringIMDB ID (например tt0111161)
search.titlestringНазвание для поиска
search.yearnumberГод (для уточнения поиска)

Условия использования

Обновлено: 1 января 2025

1. Общие положения

Используя KinoBox.tv, вы соглашаетесь с данными условиями. Если не согласны — покиньте сайт.

2. Характер сервиса

KinoBox.tv — агрегатор, встраивающий видеоплееры сторонних платформ через публичные API и iframe. Сервис не хранит и не распространяет видеофайлы. Весь контент размещён на серверах третьих сторон.

3. Возрастные ограничения

Сервис предназначен для лиц старше 18 лет.

4. Запрещённые действия

  • Попытки взлома или нарушения работы сервиса
  • Коммерческое использование без разрешения

5. Ограничение ответственности

Сервис не несёт ответственности за содержимое видеоматериалов третьих сторон и предоставляется «как есть».

Политика конфиденциальности

Обновлено: 1 января 2025

1. Что мы собираем

Только технические данные: IP-адрес (для геолокации источников), тип браузера, посещённые страницы, время визита. Персональные данные не собираются.

2. Cookies

Технические cookies для корректной работы плеера. Сторонние сервисы устанавливают свои cookies по своим политикам.

3. Сторонние плееры

Collaps, Kodik, Alloha и другие источники — независимые сервисы со своими политиками конфиденциальности.

4. Передача данных

Данные не продаются и не передаются третьим лицам, кроме случаев, предусмотренных законом.
